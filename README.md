### First time setup

```sh
git clone https://github.com/zivanovicb1/battle-simulator
```

### Installation

1.  Run `yarn` or `npm install`

### Run the simulator

1. Run `yarn start` or `npm start`  

### Tests
1. Run `yarn test` or `npm test`

1. Run `yarn run docs` or `npm run docs`
